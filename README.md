## Open App
link: https://api-restaurant-test.com

## Installation
```bash
$ "npm install"
```

## Running the app without using docker and docker compose
```bash
# development
$ "npm run start"

# watch mode
$ "npm run start:dev"

# production mode
$ "npm run start:prod"
```

## Running the app using docker and docker compose
```bash
$ "docker-compose up -d --build"
```

## Test
```bash
# unit tests
$ "npm run test"

# e2e tests
$ "npm run test:e2e"

# test coverage
$ "npm run test:cov
```
